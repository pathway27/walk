## Walk Plugin

The plugin that will (v0.1)
  - read the current TITLEID
  - read the file:
    - ux0:data/walk/TITLEID.txt
  - read position at
    - ux0:data/walk/database.txt
  - open the walkthrough at (position)
  - save positions on user action


## Building

```
mkdir build
cd build
cmake ..
make
```
